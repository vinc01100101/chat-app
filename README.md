

Simple Chat App
=================

Live app [here](https://vnze-web-chat.glitch.me) from glitch.


How to use
------------

Login:
- enter your `username` and `password`.
- press `Login` button.

Register (if you do not have account yet):
- press `Register` button.
- fill up the form.
- press `Submit`
